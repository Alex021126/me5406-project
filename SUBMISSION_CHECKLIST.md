# Submission Checklist

Before zipping the project for Canvas, make sure the final package contains:

- Source code folder with all Python modules and MuJoCo assets.
- `train.py` and `evaluate.py`.
- At least one trained model under `artifacts/models/`.
- `requirements.txt` or `environment.yml`.
- `README.md` with setup and usage instructions.
- Evaluation outputs under `artifacts/results/` if available.
- Group video file.
- Individual PDF report.

## Recommended final folder structure

```text
submission/
  code/
    artifacts/
      models/
      results/
    docs/
    src/
    README.md
    requirements.txt
    environment.yml
    train.py
    evaluate.py
  video/
    me5406_group10_demo.mp4
  report/
    your_name_report.pdf
```
